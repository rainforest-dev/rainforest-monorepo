# Initial Concept
A modern, full-stack TypeScript monorepo featuring personal projects and tools with cutting-edge web technologies.

## Target Audience
- Developers reviewing the portfolio or exploring reusable UI components.
- End-users of the personal applications and tools hosted within the monorepo.

## Key Goals
- Maintain a scalable, maintainable, and high-performance monorepo structure using Nx.
- Showcase proficiency in modern web frameworks including Next.js, Astro, and Lit.
- Provide a robust testing and continuous integration setup to easily accommodate new personal tools in the future.

## Core Projects
1. **Personal Website**: Advanced personal portfolio with AI/ML capabilities.
2. **Personal Calibre**: Recently added application for personal library/calibre management.
3. **Rainforest UI**: Modern UI component library built with Lit and Tailwind CSS.
4. **Future Personal Tools**: Expandable ecosystem designed to house various personal tools as they are developed.
*(Note: Existing projects like Personal LIFF are also maintained within the monorepo).*
