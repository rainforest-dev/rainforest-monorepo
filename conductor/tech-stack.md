# Technology Stack

## 1. Programming Language
- **TypeScript**: Used universally across the monorepo for strong typing and developer experience.

## 2. Architecture & Tooling
- **Nx**: Monorepo management, task running, and caching.
- **pnpm**: Fast, disk space efficient package manager.

## 3. Frontend Frameworks
- **Next.js**: Used for full-stack applications (e.g., Personal Calibre, Personal LIFF).
- **Astro**: Used for performance-critical, static or islands-based sites (e.g., Personal Website).
- **Lit**: Used for building reusable web components (e.g., Rainforest UI).
- **React / Vue**: Supported within the monorepo ecosystem.

## 4. UI & Styling
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **shadcn/ui**: Planned for future integration in React/Next.js projects.
- **Storybook**: Component-driven development and documentation.

## 5. Testing
- **Playwright / Cypress**: End-to-End (E2E) testing.
- **Vitest**: Fast unit testing framework.
